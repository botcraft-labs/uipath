Slack Webhook Setup:
1. Go to your Slack workspace → Apps → Create App
2. Enable "Incoming Webhooks" and add to a channel
3. Copy the Webhook URL and paste below
Telegram Bot Setup:
1. Message @BotFather → /newbot → copy your Bot Token
2. Add the bot to your channel and make it admin
3. Get your Chat ID: message @userinfobot or use https://api.telegram.org/bot<TOKEN>/getUpdates
WhatsApp via CallMeBot:
1. Add +34 644 55 64 26 to your contacts
2. Send "I allow callmebot to send me messages" to that number on WhatsApp
3. You'll receive an API key. Enter your WhatsApp number and API key below.
