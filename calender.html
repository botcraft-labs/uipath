Setup Slack Webhook:
1. Go to api.slack.com/apps → Create App → Incoming Webhooks
2. Toggle "Activate Incoming Webhooks" → Add to Workspace → Choose channel
3. Copy the Webhook URL and paste below
Setup Telegram Bot:
1. Message @BotFather → /newbot → copy Bot Token
2. Add bot to channel/group as Admin
3. Get Chat ID from @userinfobot or getUpdates
4. Channel IDs start with -100
Setup WhatsApp via CallMeBot:
1. Add +34 644 55 64 26 to contacts
2. Send: I allow callmebot to send me messages
3. You'll receive API key within minutes
4. Guide: callmebot.com →
Setup EmailJS for Invitation Emails:
1. Sign up free at emailjs.com → Dashboard
2. Email Services → Add Service (Gmail, Outlook, etc.) → Copy Service ID
3. Email Templates → Create Template → use variables below → Copy Template ID
4. Account → API Keys → Copy Public Key
5. In your template use: {{to_name}} {{to_email}} {{username}} {{password}} {{role}} {{app_url}} {{invited_by}}
ℹ️ Free EmailJS tier allows 200 emails/month. Invitation emails are sent only when you create a new user — not on edits.
