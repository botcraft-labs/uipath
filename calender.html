Setup EmailJS for Invitation Emails:
1. Sign up free at emailjs.com → Dashboard
2. Email Services → Add Service (Gmail, Outlook, etc.) → Copy Service ID
3. Email Templates → Create Template → Copy Template ID
4. ❗ CRITICAL: In template settings, set the To Email field to {{to_email}}
5. Account → API Keys → Copy Public Key
6. Template variables: {{to_name}} {{to_email}} {{username}} {{password}} {{role}} {{app_url}} {{invited_by}}
❌ The recipients address is empty
ℹ️ Free EmailJS tier allows 200 emails/month. Invitation emails are sent only when you create a new user — not on edits.
✅ Firebase is live. All browsers with this file sync in real-time.
One-time setup by admin — users need zero configuration:
1. Go to console.firebase.google.com → New Project
2. Build → Firestore Database → Create → Test mode
3. Project Settings ⚙ → Your Apps → Web → Register App → copy apiKey and projectId
4. Paste below → Save Settings → click ⬇ Download Updated HTML
5. Deploy that file to your hosting URL (Netlify / GitHub Pages / SharePoint / server)
6. Users open the URL → Firebase connects silently → they just log in. Done.
✅ Connected! Firebase is working.
🔑 How it works: When you save Firebase config and click Download Updated HTML, the Firebase credentials are embedded inside the HTML file. Anyone who opens that file auto-connects to Firebase — no setup screen, no sharing credentials separately. Just share the file.
